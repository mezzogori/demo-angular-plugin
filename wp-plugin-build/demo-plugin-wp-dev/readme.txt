Incorporate the Build Angular in WordPress via shortcode.

== Usage ==
1. Load the .zip plugin on WordPress and activate it.
2. Create a page and enter the shortcode [demo_plugin_wp_dev].
3. Recommended: Build Angular with "-Base-Href ./".
